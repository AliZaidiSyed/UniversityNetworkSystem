# University Network System Using Cisco Packet Tracer

## Overview
This project simulates a university campus network using Cisco Packet Tracer. The network consists of four departments/buildings connected through routers, switches, VLANs, and routing protocols to ensure efficient communication.

## Features
- VLAN configuration for departmental separation
- DHCP for automatic IP assignment
- RIPv2 for internal routing
- Static routing for external server access
- Cloud-hosted email server connectivity
- End-to-end network communication

## Technologies Used
- Cisco Packet Tracer
- Cisco 2911 Routers
- Cisco 3650 & 2960 Switches
- VLANs
- DHCP
- RIPv2
- Static Routing

## Departments
- Electrical & Computer Engineering
- Mathematics
- Admission Office
- Computer Science



## Outcome
Successfully designed and simulated a university network with secure communication, dynamic IP allocation, routing, and connectivity to external services.
